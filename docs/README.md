# 项目文档索引

当前项目的核心说明文档放在 `docs/` 目录下，建议按下面顺序阅读：

1. `requirements.md`
   用于说明系统目标、功能需求、非功能需求，以及与开题报告的对应关系。
2. `technical_design.md`
   用于说明项目分层结构、训练与推理链路、GUI 模块划分，以及本轮优化点。
3. `field_reference.md`
   用于说明 `configs/data`、`configs/train`、`configs/model` 中常见字段及约束。
4. `runtime_guide.md`
   用于说明本地 conda 环境下的运行、训练、GUI 启动与验证方式。
5. `compliance_and_sustainability.md`
   用于说明数据来源边界、隐私保护思路、资源消耗记录和绿色计算建议，适合论文与答辩引用。
6. `experiment_results.md`
   用于说明 2026-04-17 在 `traffic_gnn` 环境下完成的 `h3 / h6 / h12` 三模型真实对照实验结果与结论。
7. `paper_tables.md`
   用于提供可直接放入论文正文的表格模板、结果分析模板和引用文字。
8. `defense_summary.md`
   用于提供答辩结论、讲解提纲、常见问答和 PPT 组织建议。
9. `ppt_outline.md`
   用于提供答辩 PPT 的逐页标题、页面要点和可直接口述的讲稿模板。
10. `ppt_brief.md`
   用于提供 5 分钟左右答辩场景可直接使用的 8 页精简 PPT 文案。
11. `defense_qa.md`
   用于提供毕业设计现场答辩中常见问题的简洁回答模板。

建议把这十一份文档作为后续继续完善项目和准备毕业设计答辩时的基础材料。
